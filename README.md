# Bruno Ruiz | Portfolio Full Stack Developer 🚀

Bienvenido al repositorio de mi portafolio personal. Este proyecto no es solo una tarjeta de presentación, sino una demostración de código limpio, diseño accesible y arquitectura responsiva.

Analizo, construyo y optimizo soluciones basadas en lógica sólida. Comprendo la arquitectura detrás de cada línea que programo, asegurando sistemas escalables y mantenibles.

## 🛠️ Stack Tecnológico del Portafolio

* **Estructura y Estilos:** HTML5 Semántico, CSS3 (Variables personalizadas para múltiples temas).
* **Frameworks:** Bootstrap 5.
* **Interactividad:** Vanilla JavaScript (ES6+).
* **Animaciones:** AOS (Animate On Scroll) y Typewriter.js.

## ✨ Características Destacadas

* **Arquitectura de Temas Dinámicos:** Implementación de un sistema de variables CSS (`:root`) manejado mediante JavaScript para permitir al usuario alternar entre 4 temas visuales sin recargar la página.
* **Accesibilidad (a11y) 100%:** Cumplimiento de estándares WCAG. Navegación completamente funcional por teclado (Tab), contraste de colores verificado, soporte para lectores de pantalla mediante etiquetas `aria-label` y roles ARIA, e indicadores de foco (`focus-visible`).
* **Soporte Bilingüe:** Estructura replicada en inglés técnico profesional para integrarse a entornos internacionales.
* **Diseño Modular:** Layout construido priorizando *Mobile-First* y ajustado con Media Queries para una visualización perfecta en cualquier dispositivo.
